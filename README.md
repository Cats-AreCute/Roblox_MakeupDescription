# MakeupDescription
This module script allows you to create a makeup for the player using a humanoid description.

# How to install?
1. Copy the script from `installation.lua` and paste it into the command executor right here:

<img width="1920" height="66" alt="image" src="https://github.com/user-attachments/assets/07a6c139-4bc6-4b8d-a80e-e81d301817b2" />

If you can't see it, you can head to the `Script` tab and enable `Command`:

<img width="172" height="133" alt="image" src="https://github.com/user-attachments/assets/435153a3-a543-4053-aae0-0dfb2caca2c7" />

2. Run or execute the command
> **⚠️ NOTICE** `HTTPEnabled` is not necessary but it is recommended.

3. You should be able to see the module in the `game.ReplicatedStorage.Scripts`
